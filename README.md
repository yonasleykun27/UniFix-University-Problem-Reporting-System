# UniFix – University Problem Reporting System

## 🧩 Overview
UniFix is a C# desktop application that enables students and staff to report issues occurring within the university. Administrators can view, track, and resolve reported problems efficiently.

## 🏗️ Features
- User login system (Admin & Student)
- Submit new issue reports
- View report status (Pending, In Progress, Resolved)
- Admin dashboard to manage reports
- Database integration with SQL Server

## 💻 Tech Stack
- **Language:** C#
- **Framework:** .NET Framework 
- **Database:** SQL Server
- **IDE:** Visual Studio 2022
- **Version Control:** Git & GitHub
